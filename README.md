# Description
This project involves a courier delivery game where the player is responsible for transporting orders from a store to a customer location. 

# How to Run

## On Windows
Using `python version 3.11` 
```.py
py -3.11 main.py
```

## On MacOS
Using `python version 3.12`
```.py
python3 main.py
```
